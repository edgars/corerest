CoreRest Contact

CoreRest aims to be a platform for Services execution in the Google AppEngine, using Java, REST and Scripts approach.

This idea was created in order to estimulate people firgure out ways to create webservices based on dynamic languages, as well as using the REST Approach.

In the upcoming releases, CoreRest will fully support some MediaTypes as well, this is the first beta release.

Technologies

  • Integration Layer: JAX-RS Impl - JBoss RestEasy
  • Web Framework: Java Server Faces 2.0 + PrettyFaces
  • Persistence: Objectify
  • Supported Script Languages: Groovy, Ruby (upcoming: JavaScript and PHP)
