theDetails

templateInfo A tableless/fluid, two column template. Most of the dimensions are defined relative to the font size, so it gracefully changes size with different resolutions & font sizes. The code is valid XHTML 1.0 Strict & CSS.

Total size of the images and files of this template comes in at 20 kb, so it's dial-up friendly. It renders consistently in the usual suspects listed below & text only works too:

  • Firefox
  • Internet Explorer
  • Opera
  • Netscape

pictureSupport Images are fully supported by this template. You can display them using three custom classes (.imgLeft/.imgRight to align left/right & .imgBlock to create a center aligned set of images like the one below). The headings and subHeadings act as boundaries, so pictures won't overflow the section you place them in.
